# Sizzle & Drizzle

A single-page, fully client-side hot sauce shopping experience with a dark, modern aesthetic and responsive layout.

## Features

- Hero section with animated hanging chili graphics
- Product grid with images (stylized cards), descriptions, prices, and **Add to Cart** buttons
- Live cart section with quantity controls, remove buttons, and running totals
- Cart icon in the header showing the number of items
- Mock checkout form with inline validation and a fake payment confirmation message
- Dark theme, bold typography, red/orange accents, and smooth scroll between sections
- Responsive layout for desktop, tablet, and mobile

## Running the site

No build step is required. Open `index.html` in a browser:

```bash
open index.html
```

Or serve the folder with any static file server (Live Server in VS Code, `python -m http.server`, etc.) and visit the displayed URL.
